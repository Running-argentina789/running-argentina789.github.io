## 🛠 Features

The software provides several tools to improve your gaming experience.

### Adaptive Triggers
You can toggle different resistance modes for the L2 and R2 triggers. These modes simulate the mechanical feel of pulling a bowstring or pressing a heavy vehicle pedal. Select from presets like Gun, Bow, or Machine, or use the custom slider to set your own tension levels.

### Haptic Feedback
The app translates standard PC vibration into signals your controller hardware can process. You can adjust the strength of these vibrations to suit your preference.

### Light Bar Customization
Change the LED glow on your controller. Select any color using the color wheel in the settings menu. You can also turn off the lights if you prefer a darker gaming environment.

### Steam Integration
The application works well with Steam Input. Because the software emulates an Xbox controller, Steam games will recognize the input immediately. You do not need to configure complex settings inside the game menu.

## 📋 Troubleshooting

If you run into issues, check these frequent solutions:

* Controller not detected: Unplug the controller, wait five seconds, and plug it back in.
* Double input: If your game registers your controller twice, open the Controller settings in Steam. Disable PlayStation Configuration Support in the Steam menu to let this app handle the inputs.
* High latency: Use a high-quality USB cable instead of Bluetooth for the fastest response times. If you must use Bluetooth, move the adapter closer to the controller.
* Application crash: Ensure you installed the .NET Desktop Runtime from the Microsoft website. Old versions of Windows occasionally lack these required files.
